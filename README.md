## 🛠 기술 스택

- Next.js
- React Query
- TailwindCSS
- DaisyUI

## ⌨ 프로젝트 실행 방법

1. 프로젝트 저장소 복제

```bash
  git clone https://link-to-project
```

2. 폴더 이동

```bash
  cd present-client
```

3. 패키지 설치

```bash
  yarn install
  # or
  npm install
```

4. 개발 서버 시작

```bash
  yarn dev
  # or
  npm run dev
```
